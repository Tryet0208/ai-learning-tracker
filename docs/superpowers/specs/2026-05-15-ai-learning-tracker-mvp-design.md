# AI学习追踪器 MVP 设计文档

> 2026-05-15 | v1.0

## 产品定位

帮 AI 小白从不入门到能独立用 AI 工具、部署项目的学习平台。

**用户成长路径：** 小白（不会用AI） → 入门（会用AI工具） → 落地（能独立部署项目）

**目标用户：** 国内大学生/职场新人，无外网环境，预算有限，想学 AI 实战技能。

## 核心功能

### 1. 首页仪表盘

- 今日待完成任务列表（从课程模块自动生成）
- 连续打卡天数 + 本周统计
- 等级进度条（入门 → 进阶 → 独立）
- 快捷入口：继续学习、项目工坊

### 2. 学习中心

- **课程路线**：按「入门→实战→独立」三阶段组织
  - 阶段一：会用 AI 对话、Prompt 工程、环境搭建
  - 阶段二：Claude Code/Cursor 使用、Python 脚本、本地部署
  - 阶段三：理解项目结构、改配置、独立落地
- **技能模块**：每个模块 = 学（概念讲解）→ 练（课后练习）→ 测（小测验）
- **AI 问答**：学习过程中随时提问（MVP 第二阶段实现）

### 3. 项目工坊

- 精选项目列表（平台内置 + GitHub/Gitee 双源）
- 「三步落地」体验：下载资源包 → 执行一键脚本 → 打开即用
- 保姆级部署指南：中文说明、常见报错处理、截图引导
- MVP 阶段至少 1 个完整项目

### 4. 打卡系统

- 每日学习打卡（完成至少一个任务即可打卡）
- 连续天数统计 + 最长纪录
- 打卡日历视图

### 5. PWA 移动端

- 添加到手机桌面，类 App 体验
- Service Worker 离线缓存核心页面
- 响应式 UI，手机/平板/桌面自适应

## MVP 范围（第一版）

| 功能 | 内容 | 优先级 |
|------|------|--------|
| 首页仪表盘 | 今日任务、连续天数、等级进度 | P0 |
| 课程系统 | 1 条路线（入门阶段），3-5 个模块 | P0 |
| 项目工坊 | 1 个内置项目（完整部署流程） | P0 |
| 打卡系统 | 每日打卡 + 连续天数 + 日历 | P0 |
| PWA | 桌面安装 + 离线缓存 | P0 |
| 简单认证 | 账号密码登录 | P1 |
| AI 问答 | 课程内 AI 辅助 | 第二阶段 |
| 微信集成 | 小程序/公众号 | 第二阶段 |
| 付费系统 | 会员/订阅 | 第三阶段 |

## 技术架构

| 层 | 选型 | 说明 |
|----|------|------|
| 前端 | React 18 + TypeScript + Vite + Tailwind CSS 4 | 复用现有代码 |
| 后端 | Python FastAPI + SQLAlchemy | 复用现有代码 |
| 数据库 | SQLite（MVP） → PostgreSQL（后期） | 零运维起步 |
| 部署 | 前端 Vercel，后端 Railway | 免费额度起步 |
| 移动端 | PWA（manifest + Service Worker） | 无需 App 审核 |

## 数据模型（现有，无需改动）

- **User** — 用户信息、等级、学习时间、连续天数
- **LearningTask** — 学习任务（标题、类型、时长、状态）
- **Resource** — 学习资源（文章/视频，标签，难度）
- **CheckIn** — 每日打卡记录

## 不做的事（MVP）

- 微信登录/小程序
- 付费/会员体系
- 社区/评论功能
- AI 实时问答
- 桌面客户端
- 多语言
- 复杂权限系统

## 验证指标

- 10 个真实用户能完成完整学习流程
- 移动端添加到桌面后操作流畅
- 至少 1 个项目被用户成功部署到本地
